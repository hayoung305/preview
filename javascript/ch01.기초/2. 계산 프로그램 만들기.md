## 계산 프로그램 만들기
> 사칙연산을 사용

<br>

### Syntax

| 구문 | 의미 |
| ------------ | ------------- |
| + | 덧셈  |
| - | 뺄셈  |
| * | 곱셈  |
| / | 나눗셈  |
| % | 나머지  |
| ** | 제곱  |

<br>


```js

console.log(10 + 20); // 30
console.log(30 - 20); // 10
console.log(100 * 1); // 100
console.log(40 / 5); // 80
console.log(402 % 5); // 2
console.log(2 ** 3); // 8

```
